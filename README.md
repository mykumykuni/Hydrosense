# Hydrosense
Gemini said HYDROSENSE is an IoT water quality monitor for Philippine Milkfish (Chanos chanos) aquaculture. Focused on Region X small-scale hatcheries, it replaces manual testing with real-time data to prevent fish kills—which cost farmers up to 50% of annual income. We aim to modernize the ₱124B industry via the 2025-2030 ADMP national roadmap.
